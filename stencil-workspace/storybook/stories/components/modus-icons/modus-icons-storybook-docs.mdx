# Icons

---

[Modus Icon](https://modus-icons.trimble.com/modus-solid/) web components are referenced using the `<modus-icon>` custom HTML element.

### Default

<modus-icon icon="sun" color="black" size="50"></modus-icon>
<modus-icon icon="sun" color="black" size="40"></modus-icon>
<modus-icon icon="sun" color="black" size="25"></modus-icon>
<modus-icon icon="sun" color="black" size="15"></modus-icon>


### Properties

| Name          | Description                                     | Type      | Options                 | Default Value | Required |
| ------------- | ----------------------------------------------- | --------- | ----------------------- | ------------- | -------- |
| `icon`        | The name reference of the icon                  | `string`  |                         |               |          |
| `color`       | The color of the icon                           | `string` |                          |               |          |
| `size`        | The size of the icon                            | `number` |                          |               |          |

### DOM Events

| Name     | Description                      | Emits |
| -------- | -------------------------------- | ----- |
| `onClick`  | Fires when the icon is clicked  |       |
