# Icons

---

[Modus Icon](https://modus-icons.trimble.com/modus-solid/) Modus Icon component is a wrapper for icons that are replicated from https://modus-icons.trimble.com. It is referenced using the `<modus-icon>` custom HTML element.

### Default

<modus-icon icon="sun" color="black" size="50"></modus-icon>
<modus-icon icon="sun" color="black" size="40"></modus-icon>
<modus-icon icon="sun" color="black" size="25"></modus-icon>
<modus-icon icon="sun" color="black" size="15"></modus-icon>


### Properties

| Name          | Description                                     | Type      | Options                 | Default Value | Required |
| ------------- | ----------------------------------------------- | --------- | ----------------------- | ------------- | -------- |
| `icon`        | The name of the icon                            | `string`  |                         |               |          |
| `color`       | (optional) The color of the Icon                | `string`  |                         |               |          |
| `size`        | (optional) The size of the Icon                 | `number`  |                         |               |          |

### DOM Events

| Name     | Description                      | Emits |
| -------- | -------------------------------- | ----- |
| `onClick`| Fires when the icon is clicked   |       |
